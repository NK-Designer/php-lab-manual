12.Student result portal

Aim:
To create a online job portal in php with mysql.

Procedure: 
1. Start the process.
2. Create a database and create tables.
3. Create a module for entering the credentials.
4. Create a module for getting student details matching to the input.
5. Create a module for finding the result.
6. Create a module for diplaying the result.
7. Stop the process.
